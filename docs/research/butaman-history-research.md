## 1. 老祥記の歴史

### 創業者・曹松琪（そう しょうき）

- **出身地**: 中国浙江省寧波（ニンポー）
- **来日**: 1915年（大正4年）
- **妻**: 千代（日本人）
- **発想の原点**: 子どもの頃から親しんでいた天津包子（テンチンパオツー）を日本人向けにアレンジ
- **命名**: 千代さんの発案で、日本の蒸し饅頭になぞらえて「豚饅頭（ぶたまんじゅう）」と命名

### 100年以上続く「麹」

老祥記最大の資産は、**創業者が寧波から持ち込んだ麹**。

- 一般的な中華まんがイーストでふわふわに仕上げるのに対し、老祥記は**麹発酵**でもちもち食感を実現
- 酒の香りにも似た独特の風味
- 毎日発酵した麹の一部を取り分けて翌日に繋ぐ方法で、100年以上途切れることなく培養
- **神戸大空襲でも、阪神大震災でも、曹家は代々この麹を絶やすことなく守り続けた**
- 神戸の空気・気候に馴染みながら変化してきた「生きた麹」

### 曹家の系譜

| 代 | 名前 | 特記事項 |
|---|---|---|
| 初代 | 曹松琪 | 1915年創業。寧波出身。日本人妻・千代と共に豚饅頭を考案 |
| 二代目 | 曹穂昇 | 「パパ」の愛称で慕われた。戦後〜復興期に店を守る |
| 三代目 | 曹英生 | 現代表。南京町商店街振興組合理事長。KOBE豚饅サミット創設者の一人 |
| 四代目 | 曹祐仁 | 2012年より従事。フィンランド留学・電機メーカー勤務を経て家業へ |

### 老祥記と曹家包子館の関係

- 老祥記 = **「静」**（世の中に一軒しかないものを守る）
- 曹家包子館 = **「動」**（フットワークよく実験・コラボする）
- 曹家包子館は椎茸豚肉包専門で、フレンチシェフやソムリエとのコラボなど新商品開発に挑戦
- 2025年の建て替え工事中は曹家包子館の場所が老祥記の仮店舗

### 主要年表

| 年 | 出来事 |
|---|---|
| 1915年（大正4年） | 曹松琪が南京町で老祥記を創業。日本初の豚饅専門店 |
| 1945年（昭和20年） | 神戸大空襲。南京町壊滅的被害。**麹の種を守り抜く** |
| 1995年（平成7年） | 阪神淡路大震災。**再び麹を守り抜き営業再開** |
| 2011年 | KOBE豚饅サミット創設（発起人の筆頭） |
| 2025年1月31日 | 現店舗最終営業日。建て替え工事開始 |
| 2026年11月頃 | 新ビル完成予定。「次の100年に向けて」リニューアル |

### 暖簾分け: 老祥記と老祥紀

南京町には「老祥**記**」と「老祥**紀**」の一文字違いの2店が存在。元は一つの店だったが暖簾分けにより分かれた。老祥紀は花隈駅近くにある。

## 2. 神戸開港と華僑の歴史

### 1868年 神戸開港

- 慶応3年12月7日（西暦1868年1月1日）、英米の艦隊18隻から祝砲が撃たれ開港
- 横浜（1859年）より約9年遅れ
- 外国人居留地はイギリス人技師J.W.ハートが設計、22街区・126区画

### 華僑の定住

- 清国は日本と条約未締結のため、華僑は居留地に住めなかった
- 居留地の西隣に日本人と混在して居住 → これが**南京町**の原型
- 長崎から移住した中国人十数人が始まり

### 華僑コミュニティの構造

出身地別の「幇（ぱん）」と呼ばれる同郷組織:
- **広東幇**: 貿易商・料理人が多い
- **福建幇**: 福建省出身者
- **三江幇**: 江蘇・浙江・江西・安徽省など長江流域出身者（**老祥記の曹家は浙江省寧波出身でここに属する**）

### 華僑の主な職業「三刀業」

- **菜刀（さいとう）**: 料理業 → 中華料理の基礎
- **剪刀（せんとう）**: 洋服仕立て業
- **剃刀（ていとう）**: 理髪業

### 神戸華僑の規模（現在）

- 約1万人超（**日本最大**。横浜の約6,000人の約2倍）
- 神戸中華同文学校: 1899年設立、12カ国の国籍の約630名が在籍

### 他の港町との比較

| | 神戸南京町 | 横浜中華街 | 長崎新地中華街 |
|---|---|---|---|
| 華僑人口 | 約1万人超（最大） | 約6,000人 | 比較的少数 |
| 開港年 | 1868年 | 1859年 | 江戸時代 |
| 名称 | 「南京町」を現在も使用（日本で唯一） | 現在は「中華街」 | 現在は「新地中華街」 |
| 規模 | 約100店舗 | 500店舗以上 | 約30店舗 |
| 特徴 | 日本人と中国人が親密に交流する街 | 大規模観光地・商業地 | 歴史的に最古 |

## 3. 神戸と豚饅の歴史

### なぜ神戸が「豚饅の街」なのか

1. **発祥の地**: 日本初の豚饅専門店＝老祥記（1915年）
2. **老舗の集積**: 老祥記・三宮一貫楼・四興樓の「神戸三大豚まん」
3. **華僑文化との融合**: 天津包子 × 日本の醤油文化
4. **551蓬莱のルーツも神戸**: 大阪の551蓬莱（1945年創業）の創業者・羅邦強は**神戸元町の豚饅頭**にヒントを得て開発

### 「肉まん」ではなく「豚まん」と呼ぶ理由

- 関西では**「肉」＝牛肉**
- 近江牛の歴史（江戸時代の彦根藩の牛肉味噌漬け献上）
- 神戸開港で外国人が牛肉を求め、神戸ビーフが誕生
- 豚肉が入った饅頭は区別して「豚まん」と呼ぶ必要があった

### 南京町の変遷

| 時代 | 状態 |
|---|---|
| 明治〜昭和初期 | 「関西の台所」として繁栄。「南京町に行けば何でもある」 |
| 1945年 | 神戸大空襲で**全焼** |
| 戦後〜1970年代 | 闇市→外国人バーが林立する裏通りへ荒廃 |
| 1977年 | 南京町商店街振興組合設立。復活への転機 |
| 1981年 | 復興整備事業。中華門（牌楼）建設等 |
| 1995年 | 阪神大震災 → **食べ歩き文化の誕生** |
| 2011年〜 | KOBE豚饅サミット開始。全国ブランド化 |

### 震災が生んだ「食べ歩き文化」

1995年の阪神大震災で厨房が使えなくなった飲食店が、**プロパンガスとポリタンクの水**で点心・軽食のテイクアウト販売に転換。この逆境から生まれた営業形態が南京町の「食べ歩き文化」の起源となり、結果として南京町は震災後に大きな観光地へ成長した。

## 4. KOBE豚饅サミットと豚饅娘コンテスト

### KOBE豚饅サミット

- **開始**: 2011年（東日本大震災の年）
- **発起人**: 老祥記、四興樓、三宮一貫楼の3店舗
- **理念**: 「日本の元気を神戸から!」
- **開催時期**: 毎年11月11日前後
- **11月11日 = 豚饅の日**: 「11」が豚の鼻に見えることから。日本記念日協会に正式認定

### サミットの内容

- サミット限定オリジナル豚饅の販売
- 豚饅の日セレモニー（11月11日、老祥記にて）
- KOBE豚饅娘コンテスト
- 豚饅包みスピードコンテスト
- 豚饅クイズ大会（優勝者には豚饅1年分）
- スタンプラリー
- 大丸神戸店での販売イベント

### 参加店舗の推移

- 第1回（2011年）: 4店舗/団体（老祥記、四興樓、三宮一貫楼、甲南大学）
- 近年: 13〜15店舗/団体。神戸ポートピアホテル、ホテルオークラ神戸、イスズベーカリー等も参加
- 全国展開: 横浜中華街（江戸清）、長崎新地中華街（蘇州林）なども参加

### 豚饅娘コンテスト

- **開始**: 第2回サミット（2012年）から「Miss KOBE豚饅娘コンテスト」として開催
- 2024年に第13代を選出 → 第1代は2012年
- **15周年の意味**: 2012年開始なら、2026年が**15年目（第15代）**に該当

### 選考プロセス（2段階）

1. 書類審査（約10名選出）: 自己PR、豚饅に対する想い、写真
2. 最終審査（一般公開）: 神戸朝日ホール等で実施

### 活動

選出された豚饅娘は約1年間、サミットの「顔」として広報活動・イベントに参加。

### 社会貢献

- 毎年1月17日（震災の日）に豚まん無料配布
- 東日本大震災の被災地（気仙沼市等）への支援活動
- 能登半島地震への豚まん提供
- 売上の一部を被災地支援に活用

## 6. 限定豚饅コンセプト案

### 歴史 × データが示す方向性

老祥記の皮（麹発酵もちもち皮）は固定。具材で以下の空白ポジションを狙える:

### A案: 「原点回帰 -- 寧波の記憶」（餡7 × 味7）

**コンセプト**: 創業者・曹松琪の故郷・浙江省寧波の食文化に立ち返る

- 寧波は海鮮と醤油文化の街。干しエビ・干し貝柱の出汁を効かせた濃厚餡
- 醤油ベースで深みのある伝統中華味
- 111年受け継がれた麹の皮で、創業者が本当に作りたかった味を想像する
- **ストーリー**: 「15年前にサミットが始まったとき、111年前に寧波から始まった物語を思い出した。初代が海を渡って持ってきた麹で、初代の故郷の味を包む」

### B案: 「神戸の恩返し -- 震災から食べ歩きへ」（餡8 × 味6）

**コンセプト**: 1995年の震災復興から生まれた食べ歩き文化への感謝

- 神戸牛や淡路島産玉ねぎなど兵庫県産素材をふんだんに使った豪華餡
- 震災30年（2025年）+ サミット15周年の節目
- **ストーリー**: 「空襲でも震災でも守り抜いた麹。その麹が育んだ食べ歩き文化が、今度は神戸の食材で恩返しする」

### C案: 「天津から寧波、そして神戸へ」（餡7 × 味8）

**コンセプト**: 天津包子→豚饅頭への変遷を1つの豚饅で表現

- 天津の伝統的な包子の味（八角・五香粉）を控えめに取り入れつつ、醤油で仕上げる
- 中国と日本の食文化の架け橋としての老祥記の存在
- **ストーリー**: 「天津包子が海を渡り、日本人の妻・千代と出会い、豚饅頭になった。その原点の味を、15年目の今、もう一度」

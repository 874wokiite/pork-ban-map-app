# Product Overview

## Product Description
豚肉禁止マップアプリ - A web application for tracking and visualizing pork ban locations and information. The application aims to help users find areas where pork consumption is restricted or prohibited, providing geographical context and relevant details.

## Core Features
- **Interactive Map**: Display geographical areas with pork restrictions
- **Location Search**: Find specific locations and their pork ban status  
- **Information Display**: Show details about restrictions, reasons, and regulations
- **User-Friendly Interface**: Responsive design for desktop and mobile access
- **Real-time Data**: Current and accurate information about pork ban locations

## Target Use Cases
- **Travel Planning**: Travelers checking food restrictions before visiting areas
- **Dietary Compliance**: Individuals needing to avoid areas with pork-related regulations
- **Research and Education**: Academic or journalistic research on food restrictions
- **Local Business**: Restaurants and food services understanding local regulations

## Key Value Proposition
- **Comprehensive Coverage**: Single source for pork ban information across regions
- **Geographical Context**: Visual map interface for better spatial understanding
- **Accessibility**: Easy-to-use web interface accessible from any device
- **Current Information**: Up-to-date data on restrictions and regulations

## Current Status
The project is in initial development phase with a fresh Next.js foundation. The core mapping and data visualization features are yet to be implemented.
